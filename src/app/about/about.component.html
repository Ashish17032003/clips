Technology Stack

Our website is built using a powerful combination of technologies to ensure a seamless user experience. We leverage the following technologies:

  • Angular: The front-end framework that enables interactive and dynamic web pages.
  • Firebase: The backend infrastructure that provides secure storage, authentication, and real-time database functionality.
  • HTML, CSS, and JavaScript: The building blocks of web development that give life to the user interface and interactivity.
  • TypeScript: A statically typed superset of JavaScript that enhances code quality and maintainability.
  • Tailwind CSS: A utility-first CSS framework that enables rapid and consistent styling.
